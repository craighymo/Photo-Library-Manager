# Photo Library Manager

A desktop photo organization application built with JavaFX. The app allows users to create albums, add and remove photos, view photo details, and organize images with tags.

## Features

- Create, rename, and delete albums
- Add and remove photos from albums
- View photos in a desktop GUI
- Add searchable tags to photos
- Persist photo library data locally

## Technologies

- Java
- JavaFX
- Object-oriented design
- Local file/data persistence

## Status

Completed desktop application. Future improvements could include improved search, import/export tools, and cloud backup support.
